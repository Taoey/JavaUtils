# JavaUtils


### PropertiesUtil

简介：主要用于读写Properties和XML文件中的属性

|  方法   |   注释  |
| --- | --- |
|   getProperty()   |  获取某一路径下的property中的属性   |
| setProperty()    | 增加/更新一个键值                                |

    
 

### StringUtil

简介：用于简化String的一些操作

|  方法   |   注释  |
| --- | --- |
|string2File()  |    string到本地文件的转换|
|addString2File  |(追加) string到本地文件|
